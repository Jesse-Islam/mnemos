

# Mnemos 🧠

**Mnemos** is a minimal, single-page flashcard engine. No databases, no accounts, and no setup. Just a fast, focused study environment that runs entirely in your browser. everything is local, and it only stores aesthetic preference in the browser.

## 🚀 How it works

1. Open `index.html`.
2. Drop in your CSV.
3. Start learning.

## ✨ Features

* **Spaced Repetition:** Smart sorting that prioritizes cards you find difficult.
* **LaTeX Support:** High-performance math rendering via KaTeX.
* **Text-to-Speech:** Press a key to hear your cards read aloud.
* **Keyboard Driven:** Optimized for high-speed, mouse-free study sessions.
* **Aesthetic Themes:** 5 high-contrast, eye-friendly palettes with Light/Dark modes.

## 📝 CSV Format

Your file should have three columns. The first row is treated as a header and skipped.

| Question | Answer | Category |
| --- | --- | --- |
| What is the powerhouse of the cell? | Mitochondria | Biology |
| $E=mc^2$ | Mass-energy equivalence | Physics |
| "A 'string' in coding" | A sequence of characters | Programming |

## ⌨️ Controls

* <kbd>Space</kbd> : Flip Card
* <kbd>1</kbd> : **Hard** (See it again soon)
* <kbd>2</kbd> : **Easy** (See it much later)
* <kbd>3</kbd> : **Mastered** (Remove from current deck)
* <kbd>S</kbd> : Speak Text
* <kbd>Z</kbd> : Undo
* <kbd>R</kbd> : Reset Deck
